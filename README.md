# Object-Detection-Using-YOLO4
Binary class image segmentation using YOLO4. The model is able to detect Truck and Cars within an Image. <br>
The model is 90.73% accurate. <br>
Dataset :- https://www.kaggle.com/datasets/arpitgoswami47/trucks-and-cars-annotated <br>
Pre-Trained Weights :- https://drive.google.com/drive/folders/1vmSKV9arOO2nea5kHcAKi008-XwxB-Eo?usp=sharing
